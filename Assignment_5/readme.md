## Overview  
This assignment demonstrates the use of **transfer learning** for both image and text classification tasks. Pre-trained deep learning models are fine-tuned on custom datasets to perform classification efficiently.

The assignment is divided into two parts:
1. Image classification using a Convolutional Neural Network (CNN)  
2. Sentiment analysis using a Transformer model
